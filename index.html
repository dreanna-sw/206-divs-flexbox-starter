Hypertext Markup Language or HTML is the most standard language for creating Web pages and is the most basic building block of the Web. It was created by Tim-Berners Lee in the 1990s.
Cascading Style Sheets or CSS is a language used to describe how HTML elements are displayed. It is used to make web pages more presentable, with the use of colors, layouts, fonts, and etc.
Javascript is a programming language used to make web pages more interactive.
